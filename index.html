Kennarahamur

Leið fyrir þig að vinna með vefinn

1. Prófaðu vefinn

Opnaðu index.html eða keyrðu python -m http.server 8000. Smelltu á fána og skoðaðu þjóðaspjöld.

2. Breyttu gögnum

Þú getur keyrt Python API fyrir sjálfvirk úrslit. Ef API er ekki tiltækt má samt laga gögn í data/*.json.

3. Settu á Netlify

Best er að uppfæra með GitHub svo Netlify deployi sjálfkrafa. Í neyð má líka draga alla möppuna inn á Netlify Drop, en þá þarf að passa að GitHub sé uppfært líka.
