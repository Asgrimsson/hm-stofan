Hvernig nota nemendur vefinn?

Nemendur velja þjóð, lesa þjóðaspjald, skoða riðilinn og bera landið saman við Ísland. Vefurinn hentar vel í landafræði, samfélagsfræði, stærðfræði, íslensku og ensku.

Kennarahamur

Leið fyrir þig að vinna með vefinn

1. Prófaðu vefinn

Opnaðu index.html eða keyrðu python -m http.server 8000. Smelltu á fána og skoðaðu þjóðaspjöld.

2. Breyttu gögnum

Þú getur keyrt Python API fyrir sjálfvirk úrslit. Ef API er ekki tiltækt má samt laga gögn í data/*.json.

3. Settu á Netlify

Dragðu möppuna eða zip-skrána inn á Netlify Drop og þú færð slóð sem má setja í Google Sites.
